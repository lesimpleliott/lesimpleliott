## Creation du fichier `.env`

### Générer un token GitHub :

1. https://github.com/settings/tokens
2. Generate New Token (classic)
3. Scope : Repo

### Créer un fichier `.env` à la racine du projet avec le contenu suivant :

```dotenv
GITHUB_TOKEN=ghp_************
GITHUB_USERNAME=************
```
